# External pinouts

## Cooler front

9 pin D sub.

- OneWire ground
- OneWire power
- OneWire signal
- Pump ground
- Pump power
- Pump sense
- Flow sensor ground
- Flow sensor power
- Flow sensor signal

## Cooler/header tank level sensor

5 pin DIN.

- Header tank level common
- Header tank level full
- Header tank level empty

## Cooler/hoshiguma communication

5 pin DIN.

- RS485 high
- RS485 low
- RS485 ground
